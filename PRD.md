# PRD: Презентація "Правила, команди та навички для Agentic IDE"

## Огляд продукту

**Назва:** Правила, команди та навички для Agentic IDE
**Формат:** Технічна презентація (Slidev)
**Подія:** fwdays 2026
**Тривалість:** 35 хвилин

## Мета та цілі

### Головна мета
Навчити розробників ефективно використовувати та налаштовувати AI-асистентів в IDE через систему правил (rules), команд (commands) та навичок (skills).

### Цілі презентації
1. Пояснити концепцію Agentic IDE та її відмінність від традиційних інструментів автодоповнення
2. Продемонструвати практичні приклади налаштування AI-асистентів
3. Показати, як створювати власні правила, команди та навички
4. Дати учасникам готові шаблони для використання

## Цільова аудиторія

- **Основна:** Розробники програмного забезпечення (junior - senior)
- **Вторинна:** Tech leads, DevOps інженери
- **Рівень знань:** Базове розуміння роботи з IDE та AI-асистентами

### Очікування аудиторії
- Практичні приклади, які можна застосувати одразу
- Демонстрації в реальному часі
- Готові конфігурації та шаблони

## Дизайн та стиль

### Візуальний стиль
- Темна тема (developer-friendly)
- Мінімалістичний дизайн з акцентом на код
- Кольорова схема: темно-синій фон, зелені/помаранчеві акценти

### Типографіка
- Заголовки: чіткі, короткі
- Код: монохромний шрифт з підсвіткою синтаксису
- Bullet points: максимум 5 пунктів на слайд

### Анімації
- Subtle transitions (fade, slide-left)
- v-click для поетапного розкриття
- Magic Move для трансформації коду
