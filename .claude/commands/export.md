Експортуй презентацію у форматі $1.

## Кроки

1. Перейди в директорію `slidev/`
2. Виконай команду `pnpm export`
3. Результат буде в `slidev/slides-export.pdf`

## Формати

- **pdf** (за замовчуванням): Повна презентація в одному PDF
- **png**: Кожен слайд як окреме зображення

## Додаткові опції

Для PNG експорту:
```bash
pnpm export -- --format png
```

Для темної теми:
```bash
pnpm export -- --dark
```

## Перевірка

Після експорту:
1. Відкрий згенерований файл
2. Перевір всі слайди на читабельність
3. Переконайся, що код відображається коректно
