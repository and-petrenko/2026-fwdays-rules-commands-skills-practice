Порівняй поточний стан `slidev/slides.md` з `PRD.md`.

## Завдання

1. Прочитай PRD.md - секція "Структура контенту"
2. Прочитай slidev/slides.md
3. Створи таблицю відповідності

## Очікувана структура з PRD

### Секція 1: Вступ (5 хв, 4 слайди)
- 1.1 Титульний слайд
- 1.2 Про спікера
- 1.3 Що таке Agentic IDE?
- 1.4 Огляд інструментів

### Секція 2: Правила (10 хв, 7 слайдів)
- 2.1 Що таке правила?
- 2.2 Типи правил
- 2.3 Структура файлу правил
- 2.4 Приклад: Code style
- 2.5 Приклад: Архітектура
- 2.6 Live Demo
- 2.7 Best practices

### Секція 3: Команди (8 хв, 6 слайдів)
- 3.1 Що таке команди?
- 3.2 Вбудовані vs користувацькі
- 3.3 Синтаксис
- 3.4 Приклад: Code review
- 3.5 Приклад: Тести
- 3.6 Live Demo

### Секція 4: Навички (8 хв, 6 слайдів)
- 4.1 Що таке навички?
- 4.2 Команди vs Навички
- 4.3 MCP
- 4.4 Приклад: API docs
- 4.5 Приклад: CI/CD
- 4.6 Live Demo

### Секція 5: Висновки (4 хв, 4 слайди)
- 5.1 Як почати
- 5.2 Типові помилки
- 5.3 Ресурси
- 5.4 Q&A
