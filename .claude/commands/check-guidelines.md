Перевір `slidev/slides.md` на відповідність правилам контенту.

## Критерії перевірки

### 1. Дизайн слайдів
- [ ] Максимум **5 пунктів** на слайд
- [ ] Чіткі, короткі заголовки
- [ ] Немає стін тексту

### 2. Код
- [ ] Приклади **5-15 рядків**
- [ ] Є **підсвічування** рядків `{1|2|3}`
- [ ] Коментарі пояснюють код

### 3. Анімації
- [ ] `v-click` для поетапного розкриття
- [ ] Не перевантажено анімаціями
- [ ] Transitions між слайдами

### 4. Мова
- [ ] Контент **українською**
- [ ] Код/терміни **англійською**
- [ ] Консистентність термінології

### 5. Структура
- [ ] Frontmatter на кожному слайді
- [ ] Presenter notes (HTML коментарі)
- [ ] Логічний flow між слайдами

## Вивід

Створи звіт:
```
Перевірка Guidelines
====================

✓ Passed:
- [список того що OK]

⚠ Warnings:
- Слайд X: [опис проблеми]

✗ Errors:
- Слайд Y: [критична проблема]

Рекомендації:
1. [конкретна рекомендація]
```
